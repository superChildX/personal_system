# 个人成长管理系统 - 设计文档

> 为计算机专业大学生设计的一体化个人管理平台
> 
> **目标用户**: 自己（本地使用，无需部署）
> 
> **创建时间**: 2025-11-22

---

## 🎯 项目愿景

打造一个集成个人信息管理、学习成长、项目开发、生活记录于一体的个人数字空间，帮助自己更好地规划、记录和复盘大学生活与职业发展。

**核心价值**: 统一管理个人信息，快速生成简历

  - 个人资料（姓名、联系方式、邮箱、GitHub/LinkedIn等）

- **简历生成器**
  - 多模板支持（经典、现代、极简、技术向）

- **技能树可视化**
  - 技术栈雷达图

## 🛠️ 技术栈

### 后端
- **框架**: FastAPI (现代、高性能、自动API文档)
- **ORM**: SQLAlchemy (数据库操作)
- **数据库**: MySQL 8.0+
  - 账号: root
  - 密码: 123456
  - 数据库名: personal_growth_system
- **身份验证**: JWT (JSON Web Tokens)
- **API文档**: Swagger UI (FastAPI自带)
- **数据验证**: Pydantic
- **异步支持**: uvicorn + asyncio

### 前端
- **框架**: React 18+ (组件化开发，生态成熟)
- **路由**: React Router v6
- **状态管理**: Zustand / Redux Toolkit
- **UI组件库**: 
  - **shadcn/ui** (现代化、可定制)
  - **Ant Design** (企业级组件丰富)
- **样式**: TailwindCSS (快速开发，响应式设计)
- **图标**: Lucide React / React Icons
- **图表**: Recharts / Apache ECharts
- **编辑器**: 
  - Monaco Editor (代码编辑)
  - React Markdown (Markdown渲染)
  - TipTap / Lexical (富文本编辑)
- **HTTP客户端**: Axios
- **日期**: date-fns
- **构建工具**: Vite

### 数据持久化
- **数据库**: MySQL (主要数据存储)
- **文件存储**: 本地文件系统 / 云存储（可选）
- **备份**: 定期导出SQL备份

### 可选集成
- **GitHub API**: 项目数据拉取
- **Markdown解析**: remark / rehype
- **PDF生成**: jsPDF / react-pdf
- **图表可视化**: D3.js (复杂可视化)
- **思维导图**: Markmap

## 🎨 UI/UX设计原则

### 视觉风格
- **主题**: 支持亮色/暗色模式切换
- **配色**: 
  - 主色调：科技蓝 / 活力橙
  - 辅助色：中性灰阶
- **排版**: 清晰的信息层级，充足留白
- **动效**: 流畅的过渡动画，提升体验

### 交互设计
- **导航**: 侧边栏 + 顶部导航栏
- **快捷键**: 常用操作支持键盘快捷键
- **响应式**: 适配不同屏幕尺寸
- **无障碍**: 符合WCAG 2.1标准

### 用户体验
- **性能优化**: 懒加载、虚拟滚动、代码分割
- **数据安全**: HTTPS加密传输、密码哈希存储
- **反馈机制**: Toast提示、加载状态、错误处理
- **响应速度**: API缓存、数据库查询优化

## 📝 下一步行动

1. **环境准备**:
   - 安装Python 3.12+、Node.js 24+
   - 创建Conda虚拟环境：`conda create -n personal-system python=3.12`
   - 激活虚拟环境：`conda activate personal-system`
   - 安装MySQL 8.0+
   - 配置开发工具（VSCode、PyCharm等）

2. **数据库初始化**:
   - 创建数据库 `personal_growth_system`
   - 执行 `database_design.md` 中的SQL脚本
   - 验证表结构

3. **后端搭建**:
   - 初始化FastAPI项目
   - 配置数据库连接
   - 创建基础模型和API路由
   - 实现JWT认证

4. **前端搭建**:
   - 使用Vite创建React项目
   - 配置TailwindCSS
   - 创建基础组件和布局
   - 配置Axios和API服务

5. **开始开发**: 按照路线图逐步实现功能模块
